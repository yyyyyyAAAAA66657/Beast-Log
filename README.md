# 🎤 Hot Mic

> 마이크 꺼진 줄 알았지?

캐릭터는 마이크가 꺼진 줄 알지만, 사실 다 녹음되고 있습니다. RP에 **전혀 개입하지 않는** 메타 해설 확장입니다. 캐릭터는 평소처럼 행동하고, Hot Mic는 그 옆에서 다 주워 담아 자막을 답니다. 진지한 로맨스든 막장 드라마든 판타지 황제물이든, 분위기를 박살내지 않고 살짝만 긁어서 웃기게 만듭니다.

```
대사
대사
대사
─────────────
🎤 HOT MIC
[속마음 유출] 마음대로 안 했으면 좋겠음.
[제작진] 현재 본인은 매우 쿨한 모습을 연출했다고 생각 중입니다.
[팩트체크] 실패.
```

## 핵심 원칙

해설은 **단방향**입니다.

- **마이크는 세계관에 존재하지 않는다** — 캐릭터는 자기가 녹음되는 걸 모르고, 유저만 듣습니다.
- **녹음은 기억되지 않는다** — 다음 응답 생성 시 참고되지 않습니다. (백그라운드 생성, 채팅 로그에 안 남음)
- **녹음은 스토리를 바꾸지 않는다** — 사건을 추가하지 않고, 들린 것만 옮깁니다.

> RP는 마이크의 영향을 받지 않지만, 마이크는 RP를 다 듣는다.

## 3가지 나레이션 모드

| 모드 | 느낌 | 예시 |
|------|------|------|
| 🎬 **다큐멘터리** | 자연 다큐 관찰체. 캐릭터를 '개체'로 취급 | "수컷 개체는 자신을 맹수로 생각하나, 관찰 결과 대형견에 더 가깝다." |
| 🏟️ **스포츠 중계** | 캐스터 + 해설위원 핑퐁, 긴박한 실황 | "귀끝 붉어졌습니다! 시치미 작전 완전히 실패했습니다!" |
| 📺 **예능** | 한국 예능 자막. 캐릭터 체면을 4글자로 박살 | "결국 하고 싶은 말: '나 챙겨줘.' (38분째 돌려 말하는 중)" |

## 해설 구성 요소

- **속마음 유출** — 캐릭터가 말하지 않은 진심
- **제작진 코멘터리** — 행동 객관 해설
- **팩트체크** — 캐릭터 발언 검증
- **마이크에 잡힘** — 가상 인터뷰 Q&A

## 설치

SillyTavern 확장 매니저에서 아래 URL로 설치:

```
https://github.com/YOUR_USERNAME/SillyTavern-HotMic
```

또는 수동 설치: 이 폴더를 아래 경로에 복사

```
SillyTavern/public/scripts/extensions/third-party/hot-mic/
```

> **API 키 불필요.** SillyTavern에 이미 연결된 API(Claude, GPT 등)를 그대로 사용합니다. 별도 설정 없이 바로 작동합니다.

## 업데이트

확장 매니저에서 **Update** 버튼을 눌러 최신 버전을 받습니다.

> ⚠️ **업데이트가 바로 안 보이면** 브라우저가 옛 파일을 캐싱한 것입니다. **강력 새로고침**으로 해결하세요:
> - PC: `Ctrl+Shift+R` (Mac: `Cmd+Shift+R`)
> - 모바일: 브라우저 설정에서 캐시 삭제 후 새로고침
>
> 적용된 버전은 브라우저 콘솔(F12)의 `[Hot Mic] v2.x.x 로드 완료` 로그로 확인할 수 있습니다.

## 사용법

AI 응답이 생성되면 화면 하단에 자막바가 뜹니다.

- **🎤 아이콘** — 최소화 상태. 탭하면 자막바로
- **자막바** — 한 줄 미리보기. 탭하면 전체 패널로
- **전체 패널** — 모든 해설 + 모드/맥락 설정
- **↺ 버튼** — 해설 수동 재생성

### 설정

- **나레이션 모드** — 🎬 다큐 / 🏟️ 중계 / 📺 예능
- **맥락 범위** — 현재 메시지만 / 최근 5턴 / 전체 대화

## UI 상태

```
[ 🎤 ]                        최소화 (아이콘만, 빨간 REC 점)

[ 🎤 HOT MIC [🎬][최근5턴] ↺ ▼ ✕ ]
  [속마음 유출] ...               전체 패널
  [제작진] ...
  [팩트체크] ...
```
